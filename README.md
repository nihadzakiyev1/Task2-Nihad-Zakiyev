# Task2-Nihad-Zakiyev

Qeyd:Əvvəlcədən bildirmək istəyirəm ki, koddakı errorların düzəldilməsi üçün Ai dan istifadə olunubdur.Amma kodun hissələrinə nəzər yetirsəniz görəcəksinizki proyektin əsas hissəsi özüm tərəfindən hazırlanıb.
Profil şəkli və Video record Google Drive-a yükldıiyim folder-in içində mövcuddur.

Bu layihənin kod quruluşu oxunaqlı və mərhələli şəkildə təşkil olunmuşdur. HTML faylında səhifə nav, section və footer bloklarına bölünərək məntiqi ardıcıllıq yaradılmışdır. Naviqasiya menyusu səhifədaxili bölmələrə keçidi təmin edir. hero bölməsi əsas təqdimat hissəsi kimi istifadəçinin kimliyini ön plana çıxarır. about və skills bölmələri məlumatın mərhələli təqdim olunması üçün ayrıca strukturlaşdırılmışdır. Bacarıqlar hissəsində hər sahə ayrıca kartlar şəklində qurulmuşdur.

CSS faylı ümumi sıfırlama ilə başlayır və sonra komponent əsaslı şəkildə davam edir. Əvvəlcə əsas layout, daha sonra xüsusi bölmələrin stilləri yazılmışdır. Flexbox və Grid istifadə edilərək elementlərin yerləşməsi idarə olunur. Responsivlik üçün media query-lər kodun sonunda yerləşdirilmişdir.

JavaScript faylı səhifə yükləndikdən sonra işləyəcək şəkildə qurulmuşdur. Burada mobil menyu, səhifədaxili keçidlər və scroll davranışları ayrıca funksiyalarla idarə edilir. Kodun bu cür bölünməsi layihəni oxuyan şəxs üçün başa düşülməsini asanlaşdırır.
